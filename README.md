🏥 Healthcare Analytics Dashboard | Data Analytics Project

An interactive Healthcare Analytics Dashboard designed to analyze patient data, hospital performance, treatment insights, and healthcare KPIs using data visualization and analytics techniques.

🚀 Project Overview

This project focuses on healthcare data analysis to generate meaningful insights from hospital and patient-related datasets.
The dashboard helps monitor healthcare performance, patient trends, treatment analysis, and operational efficiency through interactive visualizations.

The goal is to support better healthcare decision-making using data analytics.
